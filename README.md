# 🖼 飞书图片拼图助手

飞书多维表格侧边栏插件，支持自动拼接附件图片、手动画布编辑、一键保存。

## ✨ 功能特性

- **智能网格**：自动按图片数量计算最佳布局（1张=1格，2张=2列，3-4张=2×2，5-6张=3列...）
- **多种布局**：智能网格 / 固定2列 / 固定3列 / 横排一行，自由切换
- **自动拼接**：选中含图片的行 → 自动加载 → 自动拼图 → 自动保存
- **手动画布**：支持拖拽排列、缩放、旋转、调整层级、添加文字
- **可配置**：间距、背景色、输出尺寸（1:1 / 4:3 / 16:9 / 自定义）
- **国内CDN**：使用 jsDelivr + unpkg，无需 VPN

## 📦 文件结构

```
├── index.html      # 插件主文件
├── _headers        # Cloudflare Pages 响应头配置（允许 iframe 嵌入）
└── README.md
```

## 🚀 部署到 Cloudflare Pages（推荐，国内可访问）

### 方法一：直接上传（最简单）

1. 打开 [Cloudflare Pages](https://pages.cloudflare.com/)，登录或注册账号
2. 点击 **Create a project** → **Upload assets**
3. 填写项目名称（如 `feishu-collage`）
4. 将 `index.html` 和 `_headers` 两个文件拖拽上传
5. 点击 **Deploy site**
6. 部署完成后获得访问地址：`https://feishu-collage.pages.dev`

### 方法二：通过 GitHub 部署

1. 将本项目上传到你的 GitHub 仓库
2. 在 Cloudflare Pages 选择 **Connect to Git**，关联该仓库
3. Build 设置全部留空（静态文件无需构建）
4. 点击 **Save and Deploy**

## 🔧 在飞书多维表格中添加插件

1. 打开飞书多维表格
2. 右上角点击 **插件** → **添加插件** → **自定义插件**
3. 填入 Cloudflare Pages 部署地址
4. 保存，侧边栏中即可看到插件

## 📖 使用说明

1. 确保多维表格有**两个附件字段**：一个存原图，一个存拼图结果
2. 打开插件，选择**来源字段**和**保存结果到字段**
3. 在「拼图参数」中配置布局、间距、背景色、输出尺寸
4. 点击表格中含图片的行，插件自动完成拼接并保存
5. 如需手动调整，点击「🎨 手动画布编辑」

## 📝 注意事项

- 图片临时 URL 有效期约 10 分钟，请及时完成拼接
- 建议来源字段和目标字段使用**不同**的附件字段，避免覆盖原图
- 调试阶段插件只会处理第一行记录

## 🌐 CDN 说明

本插件优先使用以下 CDN 加载飞书 JS SDK：
1. `cdn.jsdelivr.net`（国内有节点，速度快）
2. `unpkg.com`（备用）
3. `esm.run`（备用）
4. `esm.sh`（备用）
